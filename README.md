# CCTV Smart Attendance System

An AI-powered attendance management system using CCTV face recognition, liveness detection, object detection, and emotion analysis.

## Project Structure

```
cctv-attendance-system/
├── backend/          # Node.js/Express API server
├── mobile-app/       # React Native mobile application
├── admin-web/        # React admin dashboard
├── ai-models/        # AI/ML model modules
└── docker-compose.yml
```

## Tech Stack

- **Backend:** Node.js, Express, MongoDB, PostgreSQL, Redis, Socket.IO
- **Mobile:** React Native
- **Admin Web:** React
- **AI Models:** Face recognition, liveness detection, object detection, emotion analysis

## Getting Started

1. Clone the repository
2. Run `docker-compose up` to start all services
3. Install backend dependencies: `cd backend && npm install`
4. Start development server: `npm run dev`
